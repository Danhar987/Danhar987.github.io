Werkzeugverwaltung - Besonderheiten bei Postprozessoren für die Mazak INTEGREX-Serie

Vordefinition der Dreh-Werkzeuge

Für Maschinen der Mazak INTEGREX-Serie werden die Dreh-Werkzeuge bereits in der Werkzeugverwaltung mit Werkzeug-Ausrichtung und Schneiden-Nullpunkt so definiert, wie es für die Verwendung im Job nötig ist.

Im Job wird das Dreh-Werkzeug mit den, in der Werkzeugverwaltung definierten, Korrektur-Daten verwendet. Abweichende Winkel, Ausrichtungen und Schneiden-Nullpunkte können in Dreh-Jobs nicht angegeben werden, sondern müssen dürch jeweils eigene Schneidenpunkte für das selbe Werkzeug festgelegt werden.
Im Komplettwerkzeug-Aufbau können, über einen Rechtsklick auf die Schneidplatte oder auf einen bestehenden Schneidenpunkt, zusätzliche Schneidenpunkte hinzugefügt werden.

Für jeden Schneidenpunkt ist ein eigener Werkzeugkorrekturindex nötig. Dieser wird auch zur Definition der ID Code des Werkzeuges an der Steuerung verwendet.
Es ist immer nur ein freier, für die Werkzeugnummer noch nicht verwendeter, Werkzeugkorrekturindex anwählbar.
